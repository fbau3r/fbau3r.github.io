# Update
1. In den Ordner c:\tools\jekyll extrahieren
2. Mit einem `cmd` nach c:\tools\jekyll wechseln
3. `setpath.cmd` ausführen
4. `gem update jekyll` ausführen (aktuelle Version 1.5.1)
